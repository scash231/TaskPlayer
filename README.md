## Core Features
- Prev / Play-Pause / Next buttons on the taskbar
- Hooks into whatever's currently playing via Windows SMTC - Works with every application.
- Optional auto-start with Windows

## compatibility
- Windows 10 ✓
- Windows 11 ✓
- Windows 8 ??? -- and older are untested.
