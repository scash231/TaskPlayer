Windows Utility

Media controls on your taskbar.

A lightweight Windows application that integrates with System Media Transport Controls (SMTC) to display and manage active playback directly from the taskbar.

Features

  • System Integration Hooks into active media playback using Windows SMTC to control apps like Spotify, web browsers, and media players.
  • Configuration Profiles Save multiple layouts, position offsets, and startup preferences into configuration profiles.
  • Volume Control Adjust volume instantly via a hover-activated popup.
  • Auto-Hide Optionally hide control elements automatically when playback is stopped or after a configurable delay.
  • Lightweight & Native Written in C# WPF for low footprint and hardware-accelerated rendering on Windows 10 and 11.

Setup

1. Download

Get the latest zip file from the releases page.

2. Extract

Extract the files to your directory of choice.

3. Run

Run TaskbarMiniPlayer.exe. Right-click the interface to customize settings.
