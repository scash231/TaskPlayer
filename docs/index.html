Designed to feel native

TaskbarMiniPlayer builds on modern WPF concepts to bring lightweight media overlay controls directly to your Windows desktop workflow.

🎨

Dual Mode Interface

Access basic layouts instantly in Simple Mode, or toggle Expert Mode to reveal depth tweaks, alignments, performance sliders, and offset options.

💾

Savable Configuration Profiles

Create, rename, delete and transition between user profiles seamlessly. Shift between layouts, accent hues, and control speeds in one click.

⚡

Fluid Animations

Powered by custom math-driven spring easing functions. Controls open, morph, and shrink smoothly using native hardware pipelines.

Quick Setup

Get TaskbarMiniPlayer up and running on your system in under 60 seconds.

1

Download

Grab the latest executable binary release from the GitHub Releases page.

2

Unpack

Extract the archive to any directory on your computer (e.g. Program Files).

3

Launch

Run TaskbarMiniPlayer.exe. Right-click the widget to configure it.
