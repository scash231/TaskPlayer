TaskbarMiniPlayer

Windows taskbar media controller

TaskbarMiniPlayer is a lightweight Windows utility that displays and controls active media playback directly from the taskbar. It integrates with Windows System Media Transport Controls (SMTC) to support Spotify, web browsers, and other media players.

Download Latest Release • Windows 10 / 11

Features

  • Media control buttons (previous, play/pause, next) embedded in the taskbar.
  • Automatic tracking of active media players via Windows SMTC.
  • Volume control popup visible on hover.
  • Custom configuration profiles to save layouts, offsets, and options.
  • Optional auto-hide delay when playback is idle.
  • Low resource usage, built natively with C# and WPF.

Installation

  1. Download the latest release archive.
  2. Extract the files to a folder on your computer.
  3. Run TaskbarMiniPlayer.exe. Right-click the interface to configure settings.
